# SkyCast
SkyCast is a modern Weather Dashboard Web Application that provides real-time weather information for any city worldwide. It fetches live weather data using the OpenWeather API and displays detailed weather insights with a clean and responsive user interface.

# Technologies Used
HTML5
CSS3
JavaScript (ES6)
OpenWeather API

# Features
🌍 Search weather by city name
🌡 Real-time temperature display
🔄 Celsius / Fahrenheit temperature toggle
🌤 Weather condition with dynamic icons
💨 Wind speed information
💧 Humidity level
🌫 Visibility data
☁ Cloud cover percentage
🌅 Sunrise and sunset time
🕒 Last update time
☀ UV Index information
📅 5-Day Weather Forecast
📱 Fully responsive design (mobile friendly)
