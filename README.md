# Skylytics
Skylytics is a modern and interactive **Weather Dashboard Web Application** designed to provide accurate and real-time weather information for cities around the world. The application allows users to search for any city and instantly view current weather conditions using live data fetched from the **OpenWeather API**.
The dashboard presents weather information through a clean, responsive, and user-friendly interface. It displays essential weather metrics such as **current temperature, weather conditions, humidity, wind speed, visibility, pressure, UV index, cloud cover, sunrise and sunset time, and last update time**. In addition to current conditions, the application also includes a **5-day weather forecast**, enabling users to see upcoming weather trends and plan accordingly.
Skylytics also features a **Celsius and Fahrenheit temperature toggle**, allowing users to switch between temperature units easily. Weather conditions are visually represented using dynamic icons, making the interface intuitive and easy to understand. The layout is fully responsive, ensuring a smooth experience across **desktop, tablet, and mobile devices**.
By combining real-time data, modern UI design, and interactive features, Skylytics provides a simple yet powerful way for users to monitor weather conditions anywhere in the world.
